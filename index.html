시각화 프롬프트 결과
도입부 동영상 프롬프트 (Grok용)
셀팟 JSON 생성

확정된 대본을 셀팟 프로그램에 붙여넣을 JSON 텍스트로 변환합니다. 한 문장 = 한 장면 원칙. 배경(stock/ai/gif), 오버레이 텍스트, 효과음 자동 지정.

1️⃣ 셀팟에 붙여넣을 JSON

셀팟 렌더 한도 10분. 영상 길이에 따라 분할 선택 (15분→2분할, 20분→3분할, 30분→4분할).

🎬 장면별 작업 가이드 (전 장면 추천)

장면별로 이미지 3순위 추천(스톡·AI·실사 중 내용에 가장 맞는 순서)과 오버레이 추천을 표시합니다. 셀팟에서 장면 넘어가며 참고하세요.

🖼️ 이미지 리사이즈 도구 (Lambda 에러 방지)

AI 이미지 업로드 전 반드시 리사이즈하세요. 셀팟 Lambda는 고해상도 파일을 못 읽어 렌더링 에러가 납니다.
아래에 이미지 드래그 → 자동으로 1920×1080 JPG 변환 → 다운로드. 외부 사이트 갈 필요 없음.

📥
여기에 이미지 드래그 (여러 개 가능)
또는 클릭해서 파일 선택
2️⃣ 실사 이미지 교체 목록

셀팟에서 영상 생성 후, 아래 장면들만 구글에서 실제 사진으로 교체하세요. 키워드 복사 → 구글 이미지 검색 → 다운 → 셀팟에서 해당 장면 배경 교체.

3️⃣ GIF 교체 참고 목록 (Plan B)

AI 이미지가 마음에 안 드는 장면은 아래 키워드로 Klipy에서 GIF 검색해 교체할 수 있습니다. 선택 사항이므로 AI 이미지 만족스러우면 건너뛰세요.

4️⃣ 캐릭터 수동 삽입 목록 (지식세포 스타일)

정치인·국가·풍자적 설명 장면은 캐릭터를 순서대로 삽입하면 재미가 배가됩니다. 셀팟에 직접 캐릭터 이미지를 순서대로 올리세요. 선택 사항.

⚛️ Remotion 오버레이 코드 힌트 (셀팟 일괄 생성용)

핵심 임팩트 장면은 셀팟의 "AI 코드 생성 모드 → 전체 장면 일괄 생성"으로 커스텀 Remotion 애니메이션 오버레이가 자동 생성됩니다. JSON에 codeGenHint가 이미 포함되어 있어서 셀팟 AI가 참고해서 코드를 작성합니다. 별도 작업 없음.

7️⃣ AI 이미지 프롬프트 모음 (젠스파크용)

AI 이미지가 필요한 장면들의 영문 프롬프트입니다. 젠스파크에 한 번에 붙여넣기해서 무료로 이미지 생성하세요.

8️⃣ Freepik 검색어 추천 (셀팟 캐릭터 탭용)

국기·심볼·아이콘이 필요한 장면의 Freepik 검색어입니다. 셀팟 캐릭터 탭 → Freepik 검색칸에 그대로 복붙하세요.

9️⃣ 효과음 검색어 추천 (무료 사이트용)

임팩트 장면용 효과음 영문 키워드입니다. YouTube Audio Library, Freesound, Zapsplat 등 무료 사이트에 검색하세요.

🗺️ 지도 장면 (AI + 한글 라벨)

지리·경로 설명 장면은 젠스파크에 프롬프트 복붙 → 한글 라벨 박힌 지도 완성 → 셀팟 업로드. 한 번에 나오는 게 기본이고, 드물게 한글이 뭉개질 때만 캔바 백업.

🏛️ 역사 자료 몽타주 (실사 + 한글 라벨)

역사 인물 + 증거 사진을 한 화면에 모아 한글 캡션 붙인 몽타주. 구글 실사 다운 → 캔바 템플릿 → 한글 라벨 순서로 작업하세요.

5️⃣ 브랜드 로고 자동 삽입 목록 (Logo.dev)

기업명 등장 장면은 Logo.dev가 자동으로 로고를 찾아줍니다. 배경 영상은 그대로 두고, 로고만 코너에 작은 오버레이로 들어갑니다. 셀팟 AI에게 아래 목록을 전달하세요.
